# React Data Fetching Comparison
This repository demonstrates different approaches for fetching data in React applications using Fetch API, Axios and React Query as a state management.
The project was created as part of a small workshop I presented to new students in my university. The goal was to help beginners understand how modern React applications handle API communication.
During the workshop we used Chrome DevTools to analyze request timing, response behavior and overall performance in order to identify the strengths and trade-offs of each approach.

## Tech Stack
- React
- JavaScript (ES6+)
- Axios
- React Query

## Goal
Provide a simple learning reference that helps new developers understand how different data-fetching approaches work and when each one might be preferable in real applications.
